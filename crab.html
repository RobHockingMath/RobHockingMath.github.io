Crab Fractal Orb

Crab Glass Etching

Description

This vaguely crab-shaped fractal is a 3D generalization of the Kleinian limit set on page 363 of "Indra's pearls." Like many fractals in that book, it can be generated by starting with some initial circle C and a pair of Möbius transforms, a and b, as well as their inverses, A and B, and computing all possible compositions up to some fixed length using those four transforms. For example, if the length is exactly three, the possibilities are aaa, aab, aba, baa, aaB, aBa, Baa, baB, Bab... etc.
